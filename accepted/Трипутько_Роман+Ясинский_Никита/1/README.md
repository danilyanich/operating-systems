# C/C++ starter

This project contains `C/C++` starter configuration.
To start using it put source files and headers into `src` folder.

## How to use

### Generate Makefile:
```bash
cmake .
```
### Build project:
```bash
make
```

### Run project:
```bash
./bin
```
